# Klenba Popela

Klenba Popela is a premium dark fantasy promo site for a fictional 2D platformer roguelike about an exiled bell-keeper who retunes cursed bells inside a living volcanic cathedral.

## Concept

The game centers on resonance-driven runs. Each bell tone changes traversal routes, combat rhythm, and the structure of the cathedral itself.

## Website Highlights

- dark fantasy shrine-like landing page
- Czech in-site copy with atmospheric game marketing text
- custom `canvas` bell scene in `script2Dgame.js`
- dedicated design document connected to the final site

## Repository Files

- `index.html` - promo landing page
- `styles.css` - visual system and layout
- `script.js` - section interactions and tone switching
- `script2Dgame.js` - animated bell scene
- `DESING.md` - production-style design document
