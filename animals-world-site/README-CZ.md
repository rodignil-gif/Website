# Klenba Popela

Klenba Popela je prémiový dark fantasy promo web pro fiktivní 2D platformer roguelike o vyhnaném zvonopevcovi, který přelaďuje prokleté zvony uvnitř živé sopečné katedrály.

## Koncept

Základ hry stojí na rezonanci. Každý tón mění trasy, rytmus soubojů i samotnou strukturu katedrály v průběhu runu.

## Hlavní přednosti webu

- monumentální sakrální hero sekce
- veškerý viditelný obsah webu v češtině
- vlastní `canvas` scéna se zvonem v `script2Dgame.js`
- design dokument přímo navázaný na finální web

## Soubory repozitáře

- `index.html` - hlavní promo stránka
- `styles.css` - vizuální systém a layout
- `script.js` - interakce sekcí a přepínání tónů
- `script2Dgame.js` - animovaná scéna zvonu
- `DESING.md` - produkčně laděný design dokument
