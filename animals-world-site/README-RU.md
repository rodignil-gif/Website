# Klenba Popela

Klenba Popela — это премиальный dark fantasy промо-сайт для вымышленной 2D platformer roguelike игры о изгнанном звонаре, который перенастраивает проклятые колокола внутри живого вулканического собора.

## Концепт

Основа игры построена на резонансе. Каждый тон меняет маршруты, ритм боёв и саму структуру собора прямо внутри забега.

## Ключевые особенности сайта

- монументальный сакральный главный экран
- весь видимый контент сайта на чешском языке
- отдельная `canvas`-сцена с колоколом в `script2Dgame.js`
- связанный с сайтом production-style документ `DESING.md`

## Файлы репозитория

- `index.html` - промо-лендинг
- `styles.css` - визуальная система и layout
- `script.js` - интерактивность секций и переключение тонов
- `script2Dgame.js` - анимированная сцена колокола
- `DESING.md` - детальный дизайн-док проекта
