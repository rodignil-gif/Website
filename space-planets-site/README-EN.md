# Mytostroj Nadir

Mytostroj Nadir is a myth-tech sci-fi promo site for a fictional 2D platformer roguelike about Eon, a novice who flips gravity inside a collapsing orbital monastery-engine.

## Concept

The game focuses on gravity switching, polarity routes, and reading sacred machinery as both a traversal puzzle and a collapsing belief system.

## Website Highlights

- holographic HUD-inspired landing page with orbital modules
- Czech in-site copy written as a premium sci-fi indie presentation
- custom star-engine scene in `script2Dgame.js`
- design documentation that maps directly onto the site structure

## Repository Files

- `index.html` - promo landing page
- `styles.css` - myth-tech UI system and layout
- `script.js` - transitions, panels, and interactions
- `script2Dgame.js` - animated orbital engine scene
- `DESING.md` - production-style design document
