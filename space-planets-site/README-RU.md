# Mytostroj Nadir

Mytostroj Nadir — это промо-сайт и дизайн-пакет для вымышленной 2D platformer roguelike-игры о новичке звёздного монастыря, который проникает в орбитальный двигатель и перепрошивает гравитацию как часть религиозной системы.

## Концепт

Игра держится на переключении гравитации, полярных маршрутах и восприятии священной техники как traversal-головоломки и одновременно распадающейся доктрины.

## Ключевые особенности сайта

- голографический HUD-интерфейс с орбитальными модулями
- весь видимый контент сайта написан на чешском языке
- отдельная сцена звёздного двигателя в `script2Dgame.js`
- подробный `DESING.md`, напрямую связанный со структурой сайта

## Файлы репозитория

- `index.html` - главная промо-страница
- `styles.css` - myth-tech визуальная система
- `script.js` - интерактивные панели и переходы
- `script2Dgame.js` - pseudo-2D орбитальная сцена
- `DESING.md` - production-документ по дизайну
