# Mytostroj Nadir

Mytostroj Nadir je promo web a designový balík pro fiktivní 2D platformer roguelike o novicovi hvězdného kláštera, který proniká do orbitálního stroje a přepisuje gravitaci i mýtus.

## Koncept

Hra stojí na přepínání gravitace, polaritních trasách a na čtení posvátné technologie jako traversal hádanky i rozpadávajícího se náboženského systému.

## Hlavní přednosti webu

- holografické HUD rozhraní s orbitálními moduly a technickými vrstvami
- veškerý viditelný obsah webu je v češtině
- vlastní animovaná scéna hvězdného motoru v `script2Dgame.js`
- `DESING.md` přímo odpovídá finální kompozici a obsahu webu

## Soubory v repozitáři

- `index.html` - hlavní promo stránka
- `styles.css` - myth-tech vizuální systém
- `script.js` - interakce, panely a přechody
- `script2Dgame.js` - pseudo-2D orbitální scéna
- `DESING.md` - produkční designový dokument
