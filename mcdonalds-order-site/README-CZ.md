# Lucerna Dvou Měsíců

Lucerna Dvou Měsíců je panoramatický promo web pro fiktivní 2D platformer roguelike o nositelce lucerny, která přepíná mezi denní a zatměnou verzí téhož kraje.

## Koncept

Základní mechanikou je traversal mezi dvěma vrstvami světa. Jediný plamen mění cesty, chování nepřátel i pravidla prostředí během runu.

## Hlavní přednosti webu

- panoramatická kompozice s měsíční atmosférou
- veškerý viditelný marketingový text webu je v češtině
- animovaná měsíční krajina v `script2Dgame.js`
- design dokument sladěný s finálním webovým směrem

## Soubory repozitáře

- `index.html` - hlavní promo stránka
- `styles.css` - měsíční atmosféra, layout a motion
- `script.js` - přepínání světelných vrstev
- `script2Dgame.js` - animovaná měsíční scéna
- `DESING.md` - kompletní design dokument
