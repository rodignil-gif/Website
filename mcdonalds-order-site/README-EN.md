# Lucerna Dvou Měsíců

Lucerna Dvou Měsíců is a panoramic promo site for a fictional 2D platformer roguelike about a lantern-bearer who switches between daylight and eclipse versions of the same landscape.

## Concept

The core mechanic is dual-world traversal. A single flame changes paths, enemy behavior, and environmental logic across the run.

## Website Highlights

- panoramic side-scrolling mood with moonlit atmosphere
- Czech marketing copy across the whole site
- animated moon landscape in `script2Dgame.js`
- design documentation synchronized with the final web direction

## Repository Files

- `index.html` - promo landing page
- `styles.css` - moonlit atmosphere, layout, and motion
- `script.js` - light layer switching
- `script2Dgame.js` - animated moon scene
- `DESING.md` - full design document
