# Lucerna Dvou Měsíců

Lucerna Dvou Měsíců — это панорамный промо-сайт для вымышленной 2D platformer roguelike игры о носительнице фонаря, переключающей дневную и затменную версии одного и того же ландшафта.

## Концепт

Базовая механика строится на traversal между двумя слоями мира. Один единственный огонь меняет маршруты, поведение врагов и правила среды внутри run-а.

## Ключевые особенности сайта

- панорамная композиция с лунной атмосферой
- весь видимый маркетинговый текст сайта остаётся на чешском
- анимированный лунный пейзаж в `script2Dgame.js`
- дизайн-док синхронизирован с финальным веб-направлением

## Файлы репозитория

- `index.html` - промо-лендинг
- `styles.css` - лунная атмосфера, layout и motion
- `script.js` - переключение световых слоёв
- `script2Dgame.js` - анимированная лунная сцена
- `DESING.md` - полный дизайн-док
