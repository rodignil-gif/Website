# Ministerstvo Měřítek

Ministerstvo Měřítek is an archive-dossier promo site for a fictional 2D platformer roguelike about Vlasta Pečeť, a clerk who manipulates forbidden scale stamps to infiltrate a vertical bureaucratic city.

## Concept

The game is centered on changing size, bending architecture, and using administrative tools as traversal mechanics inside a living ministry that prints reality.

## Website Highlights

- dossier-style landing page with folder tabs and bureaucratic navigation
- Czech in-site copy written as a stylish indie game dossier
- custom office-machine scene in `script2Dgame.js`
- design documentation tightly connected to the final site

## Repository Files

- `index.html` - promo landing page
- `styles.css` - archival visual system and layout
- `script.js` - interactions, tabs, and reveal logic
- `script2Dgame.js` - animated ministry scene
- `DESING.md` - production-style design document
