# Ministerstvo Měřítek

Ministerstvo Měřítek je promo web a designový balík pro fiktivní 2D platformer roguelike o úředníkovi, který ukradne zakázané razítko a začne měnit velikost vlastního těla i celé architektury města.

## Koncept

Hra stojí na změně měřítka, ohýbání architektury a využívání administrativních nástrojů jako traversal mechanik uvnitř živého ministerstva, které tiskne realitu.

## Hlavní přednosti webu

- archivní dossier rozhraní se složkami, štítky a byrokratickou navigací
- veškerý viditelný obsah webu je v češtině
- vlastní animovaná kancelářsko-strojová scéna v `script2Dgame.js`
- `DESING.md` je přímo propojený s výsledným webem i celou sadou projektů

## Soubory v repozitáři

- `index.html` - hlavní promo stránka
- `styles.css` - archivní vizuální systém a layout
- `script.js` - interakce, složky a reveal efekty
- `script2Dgame.js` - pseudo-2D scéna ministerstva
- `DESING.md` - produkční designový dokument
