# Ministerstvo Měřítek

Ministerstvo Měřítek — это промо-сайт и дизайн-пакет для вымышленной 2D platformer roguelike-игры о чиновнике, который крадёт запретную печать и начинает менять масштаб собственного тела и целого вертикального города.

## Концепт

Игра построена вокруг смены размера, деформации архитектуры и использования бюрократических инструментов как traversal-механик внутри живого министерства, печатающего реальность.

## Ключевые особенности сайта

- архивный dossier-интерфейс с папками, вкладками и канцелярской навигацией
- весь видимый контент сайта написан на чешском языке
- отдельная анимированная офисно-машинная сцена в `script2Dgame.js`
- подробный `DESING.md`, связанный и с сайтом, и с общей master-таблицей

## Файлы репозитория

- `index.html` - главная промо-страница
- `styles.css` - архивная визуальная система и layout
- `script.js` - вкладки, интерактивность и reveal-эффекты
- `script2Dgame.js` - pseudo-2D сцена министерства
- `DESING.md` - production-документ по дизайну
