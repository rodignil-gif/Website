# Karneval z Nití

Karneval z Nití je plakátově pojatý promo web pro fiktivní 2D platformer roguelike o uprchlé akrobatce, která používá živou stuhu jako hák, zbraň i nástroj přežití.

## Koncept

Hra je strukturovaná jako prokletý večerní program. Každý run se mění v nový akt s odlišnými trasami, jevištní logikou i tlakem publika.

## Hlavní přednosti webu

- festivalová plakátová kompozice s asymetrickým rytmem
- veškerý viditelný text webu je v češtině
- animovaná manéž v `script2Dgame.js`
- design dokument přímo propojený s finální stránkou

## Soubory repozitáře

- `index.html` - hlavní promo stránka
- `styles.css` - plakátová typografie a vrstvený layout
- `script.js` - přepínání aktů
- `script2Dgame.js` - animovaná jevištní scéna
- `DESING.md` - kompletní design dokument
