# Karneval z Nití

Karneval z Nití — это плакатный промо-сайт для вымышленной 2D platformer roguelike игры о беглой акробатке, использующей живую ленту как крюк, оружие и инструмент выживания.

## Концепт

Игра построена как проклятая вечерняя программа. Каждый run превращается в новый акт с иной логикой сцены, маршрутов и давления публики.

## Ключевые особенности сайта

- фестивальная плакатная композиция с асимметричным ритмом
- весь видимый текст сайта оставлен на чешском языке
- анимированная манежная сцена в `script2Dgame.js`
- связанный с финальной страницей `DESING.md`

## Файлы репозитория

- `index.html` - промо-лендинг
- `styles.css` - плакатная типографика и слоистый layout
- `script.js` - переключение актов
- `script2Dgame.js` - анимированная сценическая сцена
- `DESING.md` - полный дизайн-док
