# Karneval z Nití

Karneval z Nití is a poster-driven promo site for a fictional 2D platformer roguelike about an escaped acrobat who uses a living ribbon as hook, weapon, and survival tool.

## Concept

The game is structured like a cursed evening performance. Each run becomes a new act with different traversal routes, stage logic, and audience pressure.

## Website Highlights

- festival-poster composition with asymmetrical rhythm
- Czech in-site copy and bold editorial tone
- animated carnival stage in `script2Dgame.js`
- design documentation directly linked to the shipped page

## Repository Files

- `index.html` - promo landing page
- `styles.css` - poster typography and layered layout
- `script.js` - act switching
- `script2Dgame.js` - animated stage scene
- `DESING.md` - complete design document
