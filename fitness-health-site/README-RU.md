# Pylokrev

Pylokrev — это био-органический промо-сайт для вымышленной 2D platformer roguelike игры о придворной парфюмерке, которая выживает, пересаживая себе симбиотические органы.

## Концепт

Главный hook здесь — мутация как идентичность. Каждый новый орган меняет движение, боевые действия, силуэт и общий эмоциональный тон забега.

## Ключевые особенности сайта

- асимметричный specimen-интерфейс в духе био-лаборатории
- весь видимый контент сайта остаётся на чешском языке
- анимированная тепличная сцена в `script2Dgame.js`
- подробный `DESING.md`, синхронизированный с финальной реализацией

## Файлы репозитория

- `index.html` - промо-лендинг
- `styles.css` - био-органическая визуальная система
- `script.js` - переключение мутаций и motion-логика
- `script2Dgame.js` - анимированная specimen-сцена
- `DESING.md` - полный дизайн-док
