# Pylokrev

Pylokrev is a bio-organic promo site for a fictional 2D platformer roguelike about a court perfumer who survives by grafting symbiotic organs onto her own body.

## Concept

The main hook is mutation as identity. Every new organ changes movement, combat verbs, silhouette, and the emotional tone of a run.

## Website Highlights

- asymmetrical bio-lab interface
- atmospheric Czech copy throughout the site
- animated specimen scene in `script2Dgame.js`
- detailed design document aligned with the final implementation

## Repository Files

- `index.html` - promo landing page
- `styles.css` - bio-organic visual system
- `script.js` - mutation switching and subtle motion logic
- `script2Dgame.js` - animated greenhouse specimen scene
- `DESING.md` - full design document
