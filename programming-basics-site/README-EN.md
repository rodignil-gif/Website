# Chrám Sedmi Tváří

Chrám Sedmi Tváří is a ritual codex promo site for a fictional 2D platformer roguelike about Tamar, the last young mask carver able to wear living divine masks and shift between seven movement and combat archetypes.

## Concept

The game is built around mask switching. Each divine face rewrites traversal verbs, attack timing, and the way the player reads the same temple spaces across repeated runs.

## Website Highlights

- light sacred interface inspired by parchment, carvings, and ritual wheels
- Czech in-site copy shaped like a premium indie reveal page
- custom ceremonial scene in `script2Dgame.js`
- design document directly aligned with the final web direction

## Repository Files

- `index.html` - promo landing page
- `styles.css` - ritual visual language and layout
- `script.js` - section flow, reveals, and interactions
- `script2Dgame.js` - animated shrine scene
- `DESING.md` - production-style design document
