# Chrám Sedmi Tváří

Chrám Sedmi Tváří je promo web a designový balík pro fiktivní 2D platformer roguelike o mladé řezbářce Tamar, která nosí živé božské masky a mění jimi styl pohybu, boje i vztah k prostoru.

## Koncept

Hra je vystavěná na přepínání masek. Každá tvář přepisuje traversal, rytmus útoků i způsob, jak hráč čte stejné části chrámu v dalších runech.

## Hlavní přednosti webu

- světlé rituální rozhraní inspirované pergamenem, reliéfem a obřadním kolem
- veškerý viditelný obsah webu je v češtině
- vlastní ceremoniální animovaná scéna v `script2Dgame.js`
- detailní `DESING.md` navázaný přímo na realizovaný web

## Soubory v repozitáři

- `index.html` - hlavní promo stránka
- `styles.css` - vizuální systém a layout
- `script.js` - interakce sekcí a motion logika
- `script2Dgame.js` - pseudo-2D rituální scéna
- `DESING.md` - produkční designový dokument
