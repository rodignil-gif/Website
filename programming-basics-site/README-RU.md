# Chrám Sedmi Tváří

Chrám Sedmi Tváří — это промо-сайт и дизайн-пакет для вымышленной 2D platformer roguelike-игры о молодой резчице масок Тамар, которая носит живые божественные лики и меняет ими движение, бой и ритуальную роль в мире.

## Концепт

Игра построена вокруг переключения масок. Каждое лицо переписывает traversal, темп атак и то, как игрок воспринимает одни и те же пространства храма в новых забегах.

## Ключевые особенности сайта

- светлый сакральный интерфейс с пергаментом, резьбой и ритуальным кругом
- весь видимый контент сайта написан на чешском языке
- отдельная церемониальная сцена в `script2Dgame.js`
- подробный `DESING.md`, напрямую связанный с финальным вебом

## Файлы репозитория

- `index.html` - главная промо-страница
- `styles.css` - визуальная система и layout
- `script.js` - интерактивность секций и motion-подача
- `script2Dgame.js` - pseudo-2D ритуальная сцена
- `DESING.md` - production-документ по дизайну
