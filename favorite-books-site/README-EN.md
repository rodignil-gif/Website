# Atlas Utopených Břehů

Atlas Utopených Břehů is a painterly promo site for a fictional 2D platformer roguelike about a postal cartographer who paints bridges, currents, and coastlines back into a drowned world.

## Concept

The game focuses on route-making as core traversal. Ink is both movement system and world-restoration mechanic.

## Website Highlights

- light museum-like layout with poster energy
- atmospheric Czech copy across the entire website
- animated atlas board in `script2Dgame.js`
- connected design document and master concept table

## Repository Files

- `index.html` - promo landing page
- `styles.css` - layout, palette, typography, motion
- `script.js` - card interactions and ink switching
- `script2Dgame.js` - animated atlas scene
- `DESING.md` - full design document
