# Atlas Utopených Břehů

Atlas Utopených Břehů — это painterly промо-сайт для вымышленной 2D platformer roguelike игры о почтовой картографке, которая возвращает мосты, течения и побережья обратно в затопленный мир.

## Концепт

Ядро игры строится на создании маршрутов. Чернила работают одновременно как система traversal и как механика восстановления мира.

## Ключевые особенности сайта

- светлый музейно-курaторский layout
- весь видимый текст сайта выполнен на чешском
- анимированная atlas-сцена в `script2Dgame.js`
- связанный с сайтом детальный `DESING.md`

## Файлы репозитория

- `index.html` - промо-лендинг
- `styles.css` - layout, палитра, типографика и motion
- `script.js` - интерактив карточек и переключение чернил
- `script2Dgame.js` - анимированная сцена атласа
- `DESING.md` - полный дизайн-док
