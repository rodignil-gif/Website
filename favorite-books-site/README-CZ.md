# Atlas Utopených Břehů

Atlas Utopených Břehů je malířsky laděný promo web pro fiktivní 2D platformer roguelike o poštovní kartografce, která vrací mosty, proudy a pobřeží zpět do utopeného světa.

## Koncept

Jádrem hry je vytváření tras. Inkoust funguje současně jako traversal systém i jako mechanika obnovy světa.

## Hlavní přednosti webu

- světlý layout s muzejním a kurátorským charakterem
- veškerý viditelný text webu je v češtině
- animovaná atlasová deska v `script2Dgame.js`
- design dokument přímo navázaný na finální podobu webu

## Soubory repozitáře

- `index.html` - hlavní promo stránka
- `styles.css` - layout, paleta, typografie a motion
- `script.js` - interakce karet a přepínání inkoustů
- `script2Dgame.js` - animovaná atlasová scéna
- `DESING.md` - kompletní design dokument
