# Třináctý Takt

Třináctý Takt is a pseudo-arcade promo site for a fictional 2D platformer roguelike where movement, combat, and score systems become strongest when played exactly on beat.

## Concept

The game treats rhythm as structure, not decoration. Every cassette build changes timing windows, route pressure, and the overall combat flow of a run.

## Website Highlights

- cabinet-like arcade presentation
- Czech in-site text with loud attract-mode energy
- animated equalizer scene in `script2Dgame.js`
- design document tied directly to the implemented website

## Repository Files

- `index.html` - promo landing page
- `styles.css` - acid arcade visual system
- `script.js` - track switching and score feedback
- `script2Dgame.js` - animated arcade scene
- `DESING.md` - full design document
