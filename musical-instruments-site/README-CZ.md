# Třináctý Takt

Třináctý Takt je pseudo-arkádový promo web pro fiktivní 2D platformer roguelike, kde jsou pohyb, boj i score systémy nejsilnější tehdy, když hráč hraje přesně na beat.

## Koncept

Rytmus zde není dekorace, ale struktura celé hry. Každá kazetová stavba mění timing, tlak tras i výsledný flow soubojů během runu.

## Hlavní přednosti webu

- prezentace ve stylu živého arkádového kabinetu
- veškerý text přímo na webu je v češtině
- animovaná equalizer scéna v `script2Dgame.js`
- design dokument napojený přímo na finální implementovaný web

## Soubory repozitáře

- `index.html` - hlavní promo stránka
- `styles.css` - acid arcade vizuální systém
- `script.js` - přepínání tracků a score odezva
- `script2Dgame.js` - animovaná arkádová scéna
- `DESING.md` - kompletní design dokument
