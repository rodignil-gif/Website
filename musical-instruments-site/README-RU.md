# Třináctý Takt

Třináctý Takt — это pseudo-arcade промо-сайт для вымышленной 2D platformer roguelike игры, где движение, бой и score-система достигают пика только при точной игре в ритм.

## Концепт

Ритм здесь не декор, а структура всей игры. Каждая кассетная сборка меняет тайминги, давление маршрутов и общий flow боёв в каждом run-е.

## Ключевые особенности сайта

- подача в стиле живого аркадного кабинета
- весь текст внутри сайта остаётся на чешском языке
- анимированная equalizer-сцена в `script2Dgame.js`
- дизайн-док напрямую связан с реализованной страницей

## Файлы репозитория

- `index.html` - промо-лендинг
- `styles.css` - acid arcade визуальная система
- `script.js` - переключение треков и score-отклик
- `script2Dgame.js` - анимированная аркадная сцена
- `DESING.md` - полный дизайн-док
