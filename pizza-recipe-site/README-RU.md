# Mauzoleum Pískových Žil

Mauzoleum Pískových Žil — это промо-сайт и дизайн-пакет для вымышленной 2D platformer roguelike-игры о пустынной препараторке, которая вместе с ископаемой песчаной муреной прорезает путь через живой мавзолей.

## Концепт

Игра строится вокруг разрушения окружения, туннелирования и археологических находок. Каждый забег по-новому перекраивает внутренности некрополя и открывает скрытые камеры и маршруты.

## Ключевые особенности сайта

- композиция в духе экспедиционной карты с маршрутной навигацией
- весь видимый контент сайта написан на чешском языке
- отдельная анимированная пустынная сцена в `script2Dgame.js`
- подробный `DESING.md`, напрямую связанный с финальным сайтом

## Файлы репозитория

- `index.html` - главная промо-страница
- `styles.css` - визуальная система и адаптивная вёрстка
- `script.js` - интерактивность секций и эффекты интерфейса
- `script2Dgame.js` - pseudo-2D сцена раскопок
- `DESING.md` - production-документ по дизайну
