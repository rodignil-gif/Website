# Mauzoleum Pískových Žil

Mauzoleum Pískových Žil is an expedition-map promo site for a fictional 2D platformer roguelike about Saida, a desert conservator who cuts through a living necropolis with a fossil sand moray.

## Concept

The game is built around environmental destruction, tunneling routes, and archaeological discovery. Each run redraws the interior of the mausoleum and reveals new hidden chambers.

## Website Highlights

- cartographic landing page with route-line navigation
- Czech in-site copy written as a premium indie game announcement
- custom desert motion scene in `script2Dgame.js`
- design document connected directly to the produced site

## Repository Files

- `index.html` - promo landing page
- `styles.css` - map-inspired visual system
- `script.js` - interactions, reveals, and navigation
- `script2Dgame.js` - animated excavation scene
- `DESING.md` - production-style design document
