# Mauzoleum Pískových Žil

Mauzoleum Pískových Žil je promo web a designový balík pro fiktivní 2D platformer roguelike o pouštní preparátorce, která pomocí fosilní pískové murény řeže cesty skrz pohyblivou nekropoli.

## Koncept

Hra stojí na destrukci prostředí, tunelování a archeologickém objevování. Každý run překresluje vnitřek mauzolea a otevírá nové komory, zkratky i vrstvy historie.

## Hlavní přednosti webu

- mapová kompozice s navigací vedenou trasami expedice
- veškerý viditelný obsah webu je napsaný česky
- vlastní animovaná pouštní scéna v `script2Dgame.js`
- detailní `DESING.md`, který přímo navazuje na finální web

## Soubory v repozitáři

- `index.html` - hlavní promo stránka
- `styles.css` - vizuální systém a responzivní layout
- `script.js` - interakce sekcí a efekty rozhraní
- `script2Dgame.js` - pseudo-2D scéna s pískem a trasami
- `DESING.md` - produkční designový dokument
